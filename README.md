# simple-password-generator

## Pattern Options

| Pattern Character | Description       | Example |
| ----------------- | ----------------- | ------- |
| w                 | lower case word   | word    |
| W                 | upper case word   | WORD    |
| C                 | capitalized word  | Word    |
| x                 | lower case letter | a       |
| X                 | upper case letter | A       |
| n                 | number            | 9       |
| s                 | symbol            | @       |
